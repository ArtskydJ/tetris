# tetris

> tetris clone in c

It mostly works.

## How to compile

- Get MinGW
- Run this command:

```bash
gcc -std=c99 -o Tetris.exe Tetris.c -lmingw32 -lSDLmain -lSDL -lSDL_ttf -mwindows
```

## Known issues

- The game never speeds up.
- Does not save high scores
